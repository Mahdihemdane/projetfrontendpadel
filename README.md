# Padel Court Reservation System - Developed by Mahdi Hemdane

Full-stack application for managing Padel court reservations.

## Tech Stack
- **Backend**: Spring Boot 3, Spring Security, JWT, MySQL, Hibernate.
- **Frontend**: Angular 16+, Argon Dashboard theme.
- **Database**: MySQL.

## Structure
- `/backend`: Spring Boot application.
- `/frontend`: Angular SPA.
- `/docs`: Project documentation and diagrams.

## Setup
### Backend
1. Configure `application.properties` with your database and email credentials.
2. Run `mvn spring-boot:run`.

### Frontend
1. Run `npm install`.
2. Run `npm start`.
3. Access at `http://localhost:4200`.
